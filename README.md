## Sobre el Proyecto

Este repositorio contiene el código fuente de mi portfolio personal, publicado con GitHub Pages. Reúne mi trayectoria como estudiante de Desarrollo de Aplicaciones Web (DAW), mi especialidad en backend con Java/Spring y mi proyecto final de ciclo, **EquiCampus**: un sistema de gestión para centros ecuestres con dos versiones — un monolito (Spring Boot + Thymeleaf + HTMX) y una arquitectura completa de microservicios con API Gateway, autenticación propia e integración con Odoo.

Es una página estática (HTML + CSS + JS vanilla), sin frameworks ni build step: lo que ves en el navegador es exactamente el código fuente, sin pasos de compilación intermedios.

## Instalación y Uso Local

Al ser una página estática sin dependencias, no necesita instalación. Solo clona el repositorio y abre el archivo:

```bash
git clone https://github.com/AlbGA8/AlbGA8.github.io.git
cd AlbGA8.github.io
```

Y abre `index.html` directamente en tu navegador, o sirve la carpeta con cualquier servidor estático, por ejemplo:

```bash
npx serve .
```
