Sobre mí
AG

Hola, soy Alba 👋

Soy estudiante de Desarrollo de Aplicaciones Web (DAW) con una fuerte inclinación hacia el backend. Me especializo en Java con Spring Boot, implementando arquitecturas limpias y bien estructuradas.

Mi proyecto principal es el ecosistema EquiCampus: una aplicación full-stack de gestión para centros ecuestres que existe en dos versiones — una aplicación monolítica con Spring Boot + Thymeleaf + HTMX y una arquitectura de microservicios completa con API Gateway, auth-service, servicio de gestión, servicio veterinario y frontend propio, todo orquestado con Docker Compose.

Me apasiona escribir código limpio, entender los fundamentos detrás de las tecnologías y construir soluciones que resuelvan problemas reales.

Full-Stack · Microservicios · Java · Docker

EquiCampus Microservice

Sistema de gestión integral para centros ecuestres construido con arquitectura de microservicios completa. Incluye API Gateway con Spring Cloud Gateway, auth-service independiente con JWT, servicio de gestión (alumnos, profesores, caballos, clases, pistas, pagos), servicio veterinario con ficha médica por caballo (vacunas, herrajes, medicamentos, revisiones), frontend propio y todo orquestado con Docker Compose. Integración con Odoo para gestión avanzada y librería common-lib compartida entre servicios.

Java Spring Boot Spring Cloud Gateway Spring Security · JWT Microservicios Docker Compose PostgreSQL Odoo HTML · CSS · JS JPA Thymeleaf HTMX Tailwind CSS
